Welcome to the samples environment for Microsoft Chart Control for Windows Forms. The samples environment contains a Getting Started section to get you started quickly in using the many features of the Chart control. It also contains numerous samples that demonstrate the functionalities in the Chart control and at the same time provide the code for all the samples. In the samples environment, you should be able to find the necessary code for most of the tasks you will perform with the Chart control.

Use the table of contents to see the samples in action. Each sample comes with code snippets that show you how to achieve the same effect in your Chart control. And since this environment is built directly from the code you downloaded, you can find the code for each of these samples in the directory structure.

For detailed information on the Microsoft Chart Control for .NET Framework API, download the Microsoft Chart Control for .NET Framework Documentation. For more information, join the discussions on the Microsoft Chart Forum.
